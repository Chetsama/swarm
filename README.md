# Modern Photo Gallery Website

A responsive, modern photo gallery website with filtering and lightbox functionality.

## Features

- Responsive grid layout for photos
- Category filtering (Nature, Urban, Portrait, Abstract)
- Lightbox functionality for detailed viewing
- Dark/light mode toggle
- Mobile-friendly design
- Contact form with validation

## Getting Started

1. Clone the repository
2. Navigate to the project directory
3. Run `npm start` to start the local server
4. Open your browser to `http://localhost:3000`

## Files

- `index.html` - Main webpage structure
- `styles.css` - Comprehensive styling
- `script.js` - Interactive functionality
- `server.js` - Simple Node.js server
- `package.json` - Project metadata and dependencies

## Deployment

The website is ready for deployment to any static hosting service.
