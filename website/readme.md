# Photography Portfolio Website

This is a modern photography portfolio website built with HTML, CSS, and JavaScript. It features:

- Responsive gallery layout
- Light/dark mode toggle with localStorage persistence
- Placeholder images from Unsplash
- Modern UI with smooth transitions and hover effects

## Features

1. **Responsive Design**: Adapts to different screen sizes
2. **Theme Toggle**: Switch between light and dark modes
3. **Image Gallery**: Grid layout displaying photography samples
4. **Persistent Settings**: Theme preference saved in localStorage

## Files

- `index.html`: Main HTML structure
- `styles.css`: CSS styling with dark mode support
- `script.js`: JavaScript for theme toggle and image loading
- `images/`: Folder containing 8 placeholder images

## Setup

1. Create a folder called `website`
2. Place all files in the `website` folder
3. Open `index.html` in a web browser

The site will remember your theme preference between visits.
